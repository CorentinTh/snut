## Contribute

**Pull requests are welcome !** Feel free to contribute.

To contribute to this project, first clone this repo

```bash
git clone git@github.com:cauctus/projo.git
```

Then install the dependencies

```bash
npm i
```

Then start the dev server

```bash
npm run dev
```
